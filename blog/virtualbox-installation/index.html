安裝 VirtualBox

By James 25 August 2024

前言

VirtualBox 是 Oracle 公司開發的一款免費開源虛擬化軟體，允許使用者在一台電腦上創建和運行多台虛擬機，並支援多個作業系統，如 Windows、Linux、Mac…等，在一台電腦上模擬多個獨立的運算空間。

VirtualBox 的一些主要功能：

  1. 跨平台支持：可以在 Windows、Linux、macOS 作業系统上運行。
  2. 支援多種作業系統：可以在 Windows、Linux、Mac 及其他較為小眾的作業系統。
  3. 快照功能：可以保存虛擬機的當前狀態，在需要時恢復到保存的狀態，便於測試和故障排除。
  4. 共用資料夾：允許主機作業系统和虛擬機之間共用資料夾，便於檔案傳輸。
  5. 網路配置：提供多種網路模式，如 NAT、橋接、内部網路等，模擬不同網路環境。
  6. 支持 USB：可以直接訪問實體主機的 USB 設備。

此篇文章介紹如何下載及安裝 Virtual box。

前置作業

VirtualBox 某些套件是使用 Microsoft Visual C++ 開發，所以安裝 VirtualBox 之前，主機需要先安裝 Microsoft Visual C++ 可轉發套件，依照主機 CPU 架構，下載相對應的安裝檔。

操作步驟

  1. 至 VirtualBox 官方網站，依照主機的作業系統下載相對應的安裝檔
  1. 下載完成後開啟安裝精靈
  1. 安裝內容保持預設即可，中間會有網卡更改的提醒，可能也會提醒我們缺少哪些套件，點擊「是」持續進行安裝流程即可
  1. 安裝成功畫面，點擊完成
  1. 可看到成功開啟 VirtualBox 頁面！

結語

以上為下載及安裝 VirtualBox 的流程，

VirtualBox 被廣泛應用於軟體開發、測試以及系统管理等場景中，因為它能夠提供一個隔離、安全且可控的測試和開發環境。
